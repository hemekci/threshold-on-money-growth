# Is there a threshold effect of money growth on inflation?

**Replication repository for:**

Başçı, E., Emekci, H., & Apaydın, M. O. (2025). "Is there a threshold effect of money growth on inflation?" *Applied Economics Letters*. https://doi.org/10.1080/13504851.2025.2596815

---

## Overview

This repository provides complete replication materials for analyzing threshold effects of money growth on inflation in G20 countries (1999–2023). We apply Hansen's (2017) regression kink methodology to identify non-linear monetary transmission mechanisms.

**Key Finding:** Money growth exhibits a threshold effect on inflation—the relationship changes at a specific threshold level of money growth.

## Methodology

### Regression Kink Model (Hansen 2017)

**Model Specification:**
```
y = β₁(x - γ)₋ + β₂(x - γ)₊ + β₃z + ε
```

Where:
- `y` = Deflator (inflation)
- `x` = M3LogDiff (money growth)
- `γ` = Threshold parameter (estimated)
- `z` = Control variables (lagged deflator)
- `(x - γ)₋` = min(x - γ, 0) (negative part)
- `(x - γ)₊` = max(x - γ, 0) (positive part)

**Interpretation:**
- The slope changes at the threshold γ
- Below threshold: inflation responds with slope β₁
- Above threshold: inflation responds with slope β₂
- Non-linear monetary transmission identified

**Estimation:**
- Grid search minimizes SSE over threshold parameter
- Multiplier bootstrap for inference (non-standard asymptotics)
- Numerical delta method for confidence bands

**Reference:**
Hansen, B.E. (2017). "Regression kink with an unknown threshold." *Journal of Business & Economic Statistics*, 35(2), 228-240.
